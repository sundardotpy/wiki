Apply the most specific tag that matches the user's query. Categories below map to product and process areas; use the general-enquiry tag within a category only when no more specific tag fits.

## Liquidity

| Tag | When to use |
| --- | --- |
| `Liquidity_General Enquiry` | General questions about liquidity not tied to specific investments |
| `Liquidity_Process` | Steps or requirements to liquidate assets |
| `Liquidity_Status_Update` | Updates on the status of ongoing liquidity requests |
| `Liquidity_Charges` | Pricing or discounts involved in liquidating investments |
| `Liquidity_DDPI_Status` | Updates on the status of DDPI |
| `Liquidity_cancellation` | User placed a sell order and now wants to cancel it |
| `Liquidity_Funds_Not_Received` | User has not received the sale amount after selling the bond |
