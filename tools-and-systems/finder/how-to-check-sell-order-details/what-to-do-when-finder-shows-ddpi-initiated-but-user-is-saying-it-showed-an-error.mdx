<Note>
  **Please note:** Even in cases where there is a **name mismatch** and the user encounters an error during the signing process, the Finder status may still appear as **"Initiated."**
</Note>

In such cases, kindly follow the below steps:

1. Copy the **User ID**.
2. Navigate to the **API** tab.
3. Paste the User ID under the **"Error While Signing Form"** section.
4. The API response will display the exact error, which in this case would typically be a **name mismatch**.
5. Inform the user about the name mismatch issue and request them to share a copy of their **Aadhaar card** via email at [**hello@wintwealth.com**](mailto:hello@wintwealth.com).
6. Once the Aadhaar copy is received, open Slack and navigate to the **#cx-internal-coordination** channel.
7. Raise a **KYC Workflow** request and fill in all the required details.
8. Set the appropriate expectation with the user that the issue is expected to be resolved within **1–2 working days**, subject to successful verification by the concerned team.

**Suggested user communication:**

> _"We have identified a name mismatch between your records during the signing process. Kindly share a copy of your Aadhaar card with us at_ [_hello@wintwealth.com_](mailto:hello@wintwealth.com) _so that we can coordinate with the concerned team for further verification and resolution. The issue is generally resolved within 1–2 working days after receiving the required document."_
